ACTnow Foundation - Junior Web Developer

From June 2020 - August 2020

My first professional experience came from working as a Junior developer for the ACTnow foundation. I worked alongside developers from Screamingbox LLC, to create an AngularJS web app with functionality for ordering, checking order status, and requesting free COVID-19 antibody test kits.

Platform:

Team:

A team of volunteers from all over the world collaborated together remotely in the designing, developing, testing, and deployment of the ACTnow COVID-19 teskit website.

The website built on Angular features a back end admin area with the ability to view and modify customer orders. It uses PayPal APIs to create and execute orders, and ShipBob APIs for inventory and order management. Most importantly the website has an option to grant certain people free COVID-19 test kits through our questionnaire.
